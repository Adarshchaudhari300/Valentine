# Photos Folder

Place your 30 couple photos here with the following names:

- photo1.jpg
- photo2.jpg
- photo3.jpg
- ...
- photo30.jpg

These photos will appear as falling animated background images on all Valentine's Day pages!

## Quick Setup

1. Rename your photos to photo1.jpg through photo30.jpg
2. Place them in this folder (public/photos/)
3. Rebuild: `npm run build`
4. Deploy the new `dist` folder to Netlify
